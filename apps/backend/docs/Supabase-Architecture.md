### Protected Route (With Authentication)

```
1. Client Request (with JWT) â†’ Fastify Backend
2. verifyJWT Middleware â†’ Verify Token with @fastify/jwt
3. Token Valid âœ… â†’ Decode User Info
4. fastify-supabase â†’ Create User-Scoped Client
5. Route Handler â†’ request.supabaseClient (User Client)
6. Query Database â†’ Apply RLS Policies
7. Return Data â†’ Client
```
